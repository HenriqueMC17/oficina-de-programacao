# 💻 Oficina de Programação — Laboratório de Exercícios Básicos

Este repositório reúne os exercícios de fixação, projetos práticos e laboratórios desenvolvidos durante a **Oficina de Programação**. Os projetos cobrem os fundamentos de lógica de programação, condicionais, laços de repetição e manipulação básica de elementos web (HTML, CSS e JavaScript Vanilla).

---

## 📂 Organização dos Exercícios

Abaixo está o inventário de pequenos módulos e páginas criadas nesta oficina:

*   **`calculadora/`**: Mini-calculadora interativa em JavaScript Vanilla.
*   **`pedra-papel-e-tesoura/`**: Jogo clássico de Pedra, Papel e Tesoura manipulando o estado na tela.
*   **`tabuada/`**: Gerador automatizado de tabuada utilizando laços de repetição (`for`/`while`).
*   **`idade-de-uma-pessoa/`** & **`maior-ou-menor/`**: Exercícios práticos de tomadas de decisão baseadas em condicionais (`if`/`else`).
*   **`site-urso/`**, **`site-mario-professor/`** & **`site-com-cabecalho/`**: Protótipos de landing pages e layouts estruturados em HTML5 e CSS3 Vanilla.
*   **`projeto-final/`**: Projeto integrador consolidando todas as tecnologias introduzidas.
